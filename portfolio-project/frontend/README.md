# 🌐 Annie Flora - Portfolio Project

A modern personal portfolio website built using **HTML, CSS, JavaScript, and Node.js**.  
This project showcases my learning journey as a beginner in web development.

---

## 🚀 Features

- Responsive portfolio design
- Glassmorphism UI with animations
- Typing effect in hero section
- Contact form with backend integration
- Data stored using JSON (simple database)
